## 安装

1. 安装 `node.js` [安装步骤](https://hexo.zxiaosi.cn/archives/84979429.html)

2. 安装 `pnpm` <font color="red">建议参考官网与node的对应版本</font>

3. 启动服务

   + 安装依赖 `pnpm install`
   + 运行 `pnpm run dev`

   >演示地址：http://localhost:9000/
